# 🚀 BioLink Pro - Meu Cartão de Visitas!

Projeto desenvolvido para a Fase 1 da Residência Online **IT Trends**, com foco em fundamentos de desenvolvimento web, semântica e performance extrema

## 🛠️ Tecnologias Utilizadas
- **HTML5 Semântico**: Estruturação com foco em acessibilidade e SEO.
- **CSS3 Vanilla**: Estilização pura, sem frameworks, utilizando Flexbox para layout responsivo.
- **GitHub Pages**: Hospedagem e deploy contínuo.

## 📈 Diferenciais Técnicos
- **Mobile-First**: Design planejado primeiramente para dispositivos móveis.
- **Performance**: Otimização de ativos (imagens comprimidas) para carregamento quase instantâneo.
- **Acessibilidade**: Contraste de cores validado e uso de atributos ARIA/Alt.
